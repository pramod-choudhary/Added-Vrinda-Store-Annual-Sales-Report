# Vrinda Store Data Analysis (Excel Dashboard)

## 📊 Project Overview
This project analyzes Vrinda Store’s annual sales data using Excel dashboards.  
It highlights customer demographics, regional performance, and order fulfillment efficiency.

## 🔍 Key Insights
- Women contribute 64% of total sales.
- Maharashtra and Karnataka are top-performing states.
- Amazon drives 36% of sales, followed by Myntra and Flipkart.
- Adults are the largest customer segment, with teenagers showing strong growth.

## 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data storytelling and visualization

## 🚀 Business Impact
The analysis helps identify target demographics, optimize regional campaigns, and diversify sales channels.
